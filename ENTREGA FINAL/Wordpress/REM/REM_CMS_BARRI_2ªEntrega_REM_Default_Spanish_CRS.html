6 Glosario de términos

Catálogo:Listado de una serie de productos que se ofrecen para la venta.

Cliente:Aquel usuario que se ha registrado en el sistema y, por tanto, puede acceder y comprar productos.

Usuario:Aquella persona que accede a la página web sin estar registrada en el sistema, por lo que no puede comprar productos, simplemente visualizarla.

Slider:Es un elemento de una página web que muestra múltiples imágenes y texto que se alternan entre ellas a modo de cabecera.
